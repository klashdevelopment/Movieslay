Running Movieslay Locally

  1. Go to the source link above above (Movieslay Source).
  2. Click the green "Code" button and click "Download ZIP".
  3. Make sure you have nodejs installed. Install it here:NodeJS
  4. Open your downloads folder and "extract" the zip file.
  5. Open a terminal in this folder. On windows, right-click and click "Open in Terminal" (or Command Prompt). On mac, open the Terminal app and type "cd Downloads", "cd TheFolderName".
  6. Type npm install and click enter.
  7. Type npm start and click enter.
  8. Navigate to the URL localhost:8080 in a browser, and your done. Closing the terminal will shut down Movieslay until you run the commands again.
  9. Add some movies! See below.

Adding Custom Movies

To add custom movies, you need one main things: the local MovieslayScraper program.

  1. Go to the source link above above (Movieslay Scraper Source).
  2. Click the green "Code" button and click "Download ZIP".
  3. Make sure you have nodejs installed. Install it here:NodeJS
  4. Open your downloads folder and "extract" the zip file.
  5. Open a terminal in this folder. On windows, right-click and click "Open in Terminal" (or Command Prompt). On mac, open the Terminal app and type "cd Downloads", "cd TheFolderName".
  6. Type npm install and click enter.
  7. Type start.bat on windows, or npm start on mac.
  8. Navigate to localhost:3000.
  9. In a new tab, go to torrentz2 WITH AN AD BLOCKER OR BRAVE/OPERA.
  10. Search for the movie, and click on it.
  11. Scroll down to find the player. If there is none, find another movie on torrentz2.
  12. Click the "download on Webtor" button.
  13. Copy the opened "webtor.io" link into the "webtor URL" field. Then, fill out the other items in the list.
  14. For the banner, we reccomend a rectangle size banner (more width than height).
  15. Year and IMDB are optional.

    INSTRUCTIONS FROM NOW ON ARE ABOUT UPLOADING TO LOCAL.
    If you are not, please either upload in the discord or on the movieslay site.

  16. Click "do the magic" and wait a few seconds.
  17. Click in the textbox and copy everything (CTRL-A > CTRL-C, or CMD on mac)
  18. Under the box, it should give you a file name (ex. movie-name.json).
  19. Go to your movieslay local folder.
  20. Go to public > movies.
  21. Make a new file with the name from above (ending in .json)
  22. Paste the content from the textbox into this file.
  23. Start your movieslay server, or if it was already started, refresh the page and your movie will now show up!
  24. Once done with all your movies, close the Terminal for Movieslay Scraper.
